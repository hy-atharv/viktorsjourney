# Viktor's Journey 🏞️
### A Native Android Endless Runner Built Without a Game Engine

Viktor's Journey is a Flappy Bird-inspired action arcade game built entirely from scratch in Kotlin for Android using Jetpack Compose, Compose Canvas, Koin, Android ViewModels,Kotlin Coroutines and StateFlow-based state management, without any game engine or external game framework. Featuring a custom-built game engine, the game expands upon the classic formula with weapon-based combat, enemy encounters, collectible powers, seamless map teleportation, achievements, progression systems, and dynamic gameplay mechanics that create a richer and more immersive endless adventure experience.
